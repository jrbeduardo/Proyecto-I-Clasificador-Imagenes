# Clases y Material de Estudio

Este directorio contiene material educativo sobre los conceptos fundamentales utilizados en el proyecto de clasificación de imágenes.

## Indice de Lecturas

### [Perceptrón](lecturas/perceptron.md)
Introducción al modelo más simple de neurona artificial. Cubre la definición matemática, función de activación, algoritmo de entrenamiento y limitaciones del perceptrón como clasificador binario lineal.

### [PyTorch](lecturas/pytorch.md)
Guía sobre PyTorch, biblioteca de machine learning de Meta. Incluye características principales (tensores, autograd, nn.Module), instrucciones de instalación y ejemplos de uso básico para construir redes neuronales.
